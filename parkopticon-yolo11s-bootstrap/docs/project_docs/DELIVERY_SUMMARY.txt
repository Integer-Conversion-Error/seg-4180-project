================================================================================
TECHNICAL SPECIFICATIONS
================================================================================

Algorithm:
  1. Load original and synthetic images
  2. Compute absolute difference
  3. Threshold and morphological closing
  4. Calculate coverage ratio
  5. Dilate mask to check edge touching
  6. Label connected components (scipy.ndimage)
  7. Count large components (> 100 pixels)
  8. Run validation checks
  9. Return result dict

Dependencies:
  - cv2 (OpenCV): Image I/O, differencing, morphology
  - scipy.ndimage: Connected component labeling
  - numpy: Array operations
  - All already required by project

Border Detection:
  - Dilates mask 5 iterations
  - Checks all 4 edges for pixels
  - Any non-zero pixel on edge = touch detected

Component Counting:
  - Uses scipy.ndimage.label() for CC analysis
  - Counts components with area > 100 pixels
  - Excludes background (label 0)

Integration Points:
  - Called during synthetic image processing loop
  - Only runs if: is_synthetic AND has_expected_class AND NOT --qa-skip
  - Populates manifest columns
  - Tracks failed IDs for output list
