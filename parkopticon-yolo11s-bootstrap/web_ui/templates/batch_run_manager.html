Batch Run Manager

Build nested split+train plans with forms, generate epoch sweeps, run split-only jobs, and stream live logs.

Quick Start ?

  1. Select Run Directory: choose the run folder you want to work in.
  2. Set defaults: fill split and training defaults once.
  3. Add jobs: create one or many training jobs in the Training Jobs section.
  4. Optional sweep: choose Epochs or Save Period, then enter values like `50,100,200`.
  5. Save and run: save the plan, then click Run Batch Plan.
  6. Monitor output: watch the Live Log panel for streaming CLI output.
Legend: * means required for that action. If a field is left blank, hover its ? tooltip to see what default value or fallback behavior is applied.

Default Training Configuration ?

If a training field is blank, jobs inherit batch-manager defaults: model fallback yolo11s.pt, epochs 300, save period 10, imgsz 640, device GPU 0, auto batch when unset.

Run Split Only ?

Split-only runs can be launched with most fields blank; defaults are applied automatically as described in tooltips.
