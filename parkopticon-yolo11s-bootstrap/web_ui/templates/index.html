ParkOpticon Tools

Unified interface for managing the YOLO11s dataset pipeline.

Labeler

Review bounding boxes, edit labels, and approve images for training.

Oversized Box Audit

Grid view of synthetic images with oversized boxes; audit by width/height/area and bulk-prune matched bbox entries.

Lookalike Tracker

Review class-4 lookalike negatives and tag which enforcement type they most resemble.

Synthetic Generation

Generate synthetic vehicles in empty street views for testing prompts.

Empty Review

Review and exclude images that shouldn't be used as empty backgrounds.

Synthetic Cleanup

Review bulk-generated synthetic images and delete poor quality outputs.

Synthetic Review Buckets

Review EV-SNV / EV-SE / SVO-E / SVO-NV, apply touch-ups, and bulk-approve/reject.

Training Viewer

Browse training runs in /runs/, view weights, plots, and results.

Inference Runner

Run a selected weight file on media from the UI. Defaults to root MP4 when media is left blank.

Batch Run Manager

Create split+train batch plans, run sequential experiments, and monitor logs in one place.

Map Selector

Select areas on a map to sample Street View panoramas.

Point Manager

Visualize and manage sample points from manifest CSVs.

Street View Tester

Test Street View parameters interactively.

Touchup Playground

Experiment with touchup prompts and providers on synthetic images.
